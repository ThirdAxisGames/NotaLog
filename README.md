# NotaLog
Unity Advanced Display Log System - Beta

## How to Use

````
using Logger = ThirdAxisGames.NotaLog;


public void Start(){
    Logger.Log("Hello");
    Logger.CriticalLog("I'am");
    Logger.WarningLog("NotaLog");
    }

````
